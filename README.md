Mardizza UI Kit
===============

Templates, style guides and style sheets for Mardizza project

## Dependencies

* Nodejs (npm)
* Gulp (npm install gulp -g)

## Installation
* git clone https://github.com/sergemazille/mardizza_ui_kit.git
* cd mardizza_ui_kit
* npm install
* gulp deploy
